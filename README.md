# Sierpinski-Gasket
Generating Sierpinski Triangle (fractal shape) with OpenGL

### Clone & Run
```bash
git clone https://github.com/OmarAzizi/Sierpinski-Gasket.git && cd Sierpinski-Gasket
g++ -o sierpinski main.cpp -lGL -lGLU -lglut && ./sierpinski
```
